This script creates a new Electron project with the following features:

- TypeScript
- Vite
- Vue 3
- Pinia
- Vue Router
- Tailwind CSS
- ESLint (including the Vue plugin)
- Prettier
- VSCode settings

## Dependencies

- `npm` command
